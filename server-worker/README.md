# HOW TO RUN
```
npm install
```
---
## .env
Create an .env with the same properties as the .env.example.

If you wanted to run locally (without redis), start the docker container with:
 ```
npm run compose:up
```
and leave the connection url as default.

---
## Run
#### DEV
```
 npm run dev
```

#### Deploy
```
npm run start
```
